Project Description:
This is a web development project titled "Welcome to Pakistan"created for 1st semester.  
The website showcases the beautiful tourist destinations, culture, and attractions of Pakistan.  
It is designed using HTML, CSS, and JavaScript to provide an interactive and visually appealing experience for visitors.  

Purpose:
- To highlight the top tourist destinations across Pakistan.  
- To create an engaging platform for users to explore Pakistani culture and landmarks.  
- To practice web development skills as part of our semester project.  

Features:  
- Interactive pages for different regions of Pakistan  
- Responsive design for desktop and mobile  
- Image galleries and information sections about famous places  

Semester: 1st Semester  
Course: Web Development / ICT Project
